Web usability

There are a lot of factors that affect web usability, the main things affecting it being its multimedia elements

When creating webpages, we have to consider how we want to create and place media elements such as text, images, sound, video, and etc.

We need to account for everyone, meaning the physically impaired as well as everyone else.

So when we create new elements and add it to the web, we need to stop and think; Is this a good design?
